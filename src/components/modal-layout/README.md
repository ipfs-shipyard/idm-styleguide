# Modal Layout

A standard layout component for modal.

## Usage

```jsx
import { ModalLayout } from '@nomios/web-uikit';

<ModalLayout variant="half-panel-bordered">
    <div>Your content</div>
    <div>Goes here</div>
</ModalLayout>
```
